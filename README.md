# Intro To Computer Vision Final Project

Welcome to the Introduction to Computer Vision Final Project repository. This project demonstrates fundamental computer vision concepts and techniques.

## 📋 Project Overview

This repository contains the final project for an Introduction to Computer Vision course. The project applies core computer vision principles to solve practical problems using image processing and analysis.

## 🎯 Objectives

- Apply fundamental computer vision concepts
- Implement image processing algorithms
- Analyze and interpret visual data
- Develop practical computer vision solutions

## 🛠️ Technologies & Tools

- **Python** - Primary programming language
- **OpenCV** - Computer vision library
- **NumPy** - Numerical computing
- **Matplotlib** - Data visualization
- **Jupyter Notebooks** - Interactive development

## 🚀 Getting Started

### Prerequisites

- Python 3.7 or higher
- pip package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/AriooGN/Intro-To-Computer-Vision-Final-Project.git
cd Intro-To-Computer-Vision-Final-Project
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```
